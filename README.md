<h1>How to use:</h1><br>
1) update the database configuration file:
<pre>
nano db.conf
</pre>
2) start the server with:
<pre>
sudo newlisp -http -d 80 -w ~/path/to/newstrap/
</pre>
3) point your browser to:
<pre>
http://localhost/index.cgi
</pre>
