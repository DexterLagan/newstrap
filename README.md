# newstrap
A fast, lightweight web framework written in newLISP

How to use:
1) start the server with:
<pre>
sudo newlisp -http -d 80 -w ~/Code/newlisp/newstrap/
</pre>
2) point your browser to:
<pre>
http://localhost/index.cgi
</pre>
